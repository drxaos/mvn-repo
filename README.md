# mvn-repo

## install repo
```xml
    <repositories>
        <repository>
            <id>drxaos</id>
            <name>mvn-repo</name>
            <url>https://github.com/drxaos/mvn-repo/raw/master/</url>
        </repository>
    </repositories>
```

## mmx-web
https://github.com/drxaos/mmx-web
```xml
    <dependencies>
        <dependency>
            <groupId>com.github.drxaos</groupId>
            <artifactId>mmx-web</artifactId>
            <version>1.1</version>
        </dependency>
    </dependencies>
```
